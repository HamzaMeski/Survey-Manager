# 📊 Survey Manager (Backend APIs: Spring Boot 3.2.3)

A robust Spring Boot application for managing surveys, questions, and answers with comprehensive analytics capabilities.

## 🌟 Features

### 🔐 Authentication & Authorization
- JWT-based authentication
- Role-based access control (Admin/User)
- Secure password hashing
- Token refresh mechanism

### 📝 Survey Management
- Create, read, update, and delete surveys
- Survey categorization
- Survey status tracking (Draft/Published/Closed)
- Survey expiration dates

### ❓ Question Management
- Multiple question types:
  - Single Choice
  - Multiple Choice
- Question ordering
- Required/Optional questions
- Question grouping

### ✅ Answer Management
- Answer validation
- Response tracking
- Answer statistics

### 📊 Analytics
- Response rate tracking
- Answer distribution
- User participation statistics

## 🛠️ Technical Stack

### Backend
- Java 17
- Spring Boot 3.2.3
- Spring Security
- Spring Data JPA
- PostgreSQL
- Lombok
- MapStruct

### Testing
- JUnit 5
- Mockito

### Documentation
- OpenAPI (Swagger)
- JavaDoc
- Postman Collection

## 🚀 Getting Started

### Prerequisites
- Java 17+
- Maven 3.6+
- PostgreSQL 12+
